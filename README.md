## Описание
Консольное приложение для управления простой базой данных с поддержкой CRUD операций.

## Установка и запуск

1. Клонировать репозиторий
2. Установить зависимости: `poetry install`
3. Запустить базу данных: `poetry run database`

## CRUD-операции

### Основные команды:

## Создание записи:
insert into <имя_таблицы> values (<значение1>, <значение2>, ...)
Пример: `insert into users values ("Sergei", 28, true)`

## Чтение записей:
select from <имя_таблицы> [where <столбец> = <значение>]
Примеры:
- `select from users` (все записи)
- `select from users where age = 28` (с фильтром)

## Обновление записей:
update <имя_таблицы> set <столбец> = <новое_значение> where <условие>
Пример: `update users set age = 29 where name = "Sergei"`

## Удаление записей:
delete from <имя_таблицы> where <условие>
Пример: `delete from users where ID = 1`

## Информация о таблице:
info <имя_таблицы>

**Примечание**: Условия WHERE поддерживают только оператор `=`.

## Управление таблицами

## Команды:
- `create_table <имя_таблицы> <столбец1:тип> <столбец2:тип> ...` - создать таблицу
- `list_tables` - показать список всех таблиц
- `drop_table <имя_таблицы>` - удалить таблицу

## Поддерживаемые типы данных:
- `int` - целые числа
- `str` - строки (в кавычках: "текст")
- `bool` - логические значения (true/false)

## Пример использования:

```bash

| ID |  name  | age | is_active |
+----+--------+-----+-----------+
| 1  | Sergei | 28  |    True   |
+----+--------+-----+-----------+

update users set age = 29 where name = "Sergei"
Записи в таблице "users" успешно обновлены. Обновлено записей: 1

select from users where name = "Sergei"
+----+--------+-----+-----------+
| ID |  name  | age | is_active |
+----+--------+-----+-----------+
| 1  | Sergei | 29  |    True   |
+----+--------+-----+-----------+

delete from users where ID = 1
Записи успешно удалены из таблицы "users". Удалено записей: 1

info users
Таблица: users
Столбцы: ID:int, name:str, age:int, is_active:bool
Количество записей: 0
```
